# Toolie UI Mock

A mock prototype of the Toolie UI with some limited functionality.

## Getting Started
Clone this repo to your local enviornment.

```
To run on Mac / Linux
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) with your browser.
